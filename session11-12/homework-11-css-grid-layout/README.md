# 🎨 Photo Gallery Layouts (CSS Grid)

[დავალება](https://www.figma.com/design/WomErT1ndT5KKfX5zd4SdM/Grid-Layout---Photo-gallery?node-id=0-1&t=bBFmVn7cg8WS3TmY-1)

ამ დავალების მიზანია **CSS Grid** ტექნოლოგიის გამოყენებით ოთხი სხვადასხვა ფოტო გალერეის ლეიაუთის აწყობა. თითოეული დიზაინი მოითხოვს ბადის (Grid) სტრუქტურის სწორად დაგეგმვას და ელემენტების განლაგებას.

## 📌 ძირითადი მოთხოვნები

1.  **ტექნოლოგია:** ლეიაუთები უნდა აიწყოს ექსკლუზიურად **CSS Grid**-ის გამოყენებით.
    
2.  **სტრუქტურა:** თითოეული ლეიაუთისთვის (MacBook Pro 14" - 1, 3, 5, 6) შექმენით ცალკეული HTML ფაილი (მაგ: `layout1.html`, `layout2.html` და ა.შ.) ან გამოიყენეთ ერთი მთავარი გვერდი შესაბამისი სექციებით.
    
3.  **ვიზუალური სიზუსტე:** საბოლოო შედეგი მაქსიმალურად უნდა ჰგავდეს მოწოდებულ Figma-ს დიზაინს (Padding, Gap, Alignment).
    
4.  **კოდის ხარისხი:**
    -   გამოიყენეთ სემანტიკური HTML თეგები.
    -   დაწერეთ სუფთა და ორგანიზებული CSS.
    -   დაიცავით ინდენტაცია (Naming conventions).
        

## 🚀 სამუშაო პროცესი (Workflow)

-   **Commits:** მნიშვნელოვანია პროცესის ჩვენება. აკეთეთ ხშირი და აზრიანი "Commit"-ები (მაგ: `feat: add grid structure for layout 1`, `style: update spacing for gallery`).
-   **Images:** გამოიყენეთ დიზაინში მოცემული ფოტოები ან მსგავსი Placeholder სურათები (მაგალითად: [Unsplash](https://unsplash.com/) ან [Pexels](https://pexels.com/)).
